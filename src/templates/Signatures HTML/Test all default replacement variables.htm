Signature template file containing all default replacement variables

Remarks

This signature template file contains all default replacement variables defined in '.config\default replacement variables.ps1'.

As this signature template file is not generated automatically, it may not be up to date. Should you find an error, please create a GitHub ticket at https://github.com/GruberMarkus/Set-OutlookSignatures/issues.

If you have created custom replacement variables, these variables are not automatically included here.
